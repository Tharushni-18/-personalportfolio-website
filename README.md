1. Project Overview

A Personal Portfolio Website is a responsive web application designed to showcase personal information, technical skills, projects, certifications, resume, and contact details in a professional manner. The website is developed using HTML, CSS, and JavaScript with a modern and user-friendly interface that works on both desktop and mobile devices. 
It helps create a strong online presence and allows recruiters or employers to easily view the user’s achievements and work. 
This project also demonstrates front-end web development skills, responsive web design, and creativity in website design.


2. Features

* Responsive and modern design
* Home/Landing page
* About Me section
* Skills showcase
* Projects portfolio
* Resume/CV section
* Contact form
* Social media links
* Smooth scrolling and animations



3.Technologies Used

* HTML5

* JavaScript
* Google Fonts
* Responsive Web Design
